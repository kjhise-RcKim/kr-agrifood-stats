# 배포 가이드 — AI 챗봇 켜기 (Vercel + Gemini 무료)

> 대시보드·검색·자동분석은 지금도 `web/index.html`을 더블클릭하면 작동합니다.
> **AI 챗봇만** 키를 숨길 서버가 필요해서 Vercel에 한 번 올려야 켜집니다. (전부 무료)

## 준비물 (2개)
1. Google 계정 (Gemini 키 발급용)
2. GitHub 계정 (이미 있음 — 저장소 연결됨)

---

## 1단계. Gemini 무료 API 키 발급 (약 2분)
1. 브라우저에서 **aistudio.google.com** 접속 → 구글 계정 로그인.
2. 왼쪽/상단의 **Get API key** 클릭 → **Create API key** 클릭.
3. 새 프로젝트 선택(또는 생성) → 만들어진 **키 문자열을 복사**해 메모장 등에 잠깐 보관.
   - 신용카드 불필요. 우리 데이터는 공개 통계라 무료 티어로 충분합니다.
   - ⚠️ 이 키는 비밀번호처럼 취급 — 아무 데나 붙여넣지 마세요. (곧 Vercel에만 넣습니다)

## 2단계. Vercel에 배포 (약 5분)
1. **vercel.com** 접속 → **Sign Up** → **Continue with GitHub**로 로그인.
2. **Add New… → Project** → 목록에서 `kr-agrifood-stats` 저장소 **Import**.
3. 설정 화면에서 **딱 두 가지**만 바꿉니다:
   - **Root Directory**: `Edit` 눌러 **`web`** 선택. (중요 — 이래야 사이트와 `/api/chat`가 함께 잡힘)
   - **Environment Variables**: 아래 한 줄 추가
     - Name: `GEMINI_API_KEY`
     - Value: 1단계에서 복사한 키 붙여넣기
   - (선택) 모델 바꾸려면 `GEMINI_MODEL` = `gemini-2.5-flash` 추가. 기본값이라 안 넣어도 됨.
4. **Deploy** 클릭 → 1~2분 기다리면 `https://kr-agrifood-stats-xxxx.vercel.app` 주소가 나옵니다.

## 3단계. 확인
- 배포된 주소를 열고, 맨 아래 **💬 AI에게 물어보기**에 질문 입력.
  - 예: "농가인구와 고령화율의 관계는?", "식량자급률은 왜 낮아?"
- 답변에 출처(장·페이지)가 붙고, 데이터에 없는 값은 "자료에 없습니다"라고 답하면 정상.

## 참고
- **비용**: 호스팅(Vercel)·API(Gemini 무료 티어) 모두 무료. 무료 한도(대략 분당 10~15회)면 개인·소규모 공개용엔 충분.
- **키 안전**: 키는 Vercel 서버 환경변수에만 있고, 웹페이지 소스에는 절대 노출되지 않습니다(`web/api/chat.js`가 대신 호출).
- **키 교체**: 키를 바꾸려면 Vercel → 프로젝트 → Settings → Environment Variables에서 값만 수정 후 재배포.
- **자동 갱신**: GitHub Desktop으로 Push할 때마다 Vercel이 자동으로 다시 배포합니다.
- **한도 초과/오류 시**: 챗 답변창에 오류 메시지가 그대로 표시됩니다(예: 429 = 잠시 후 재시도).

## GitHub Pages는?
챗봇 없이 대시보드만 공개하려면 GitHub Pages도 가능하지만, 챗봇(서버 필요) 때문에 **Vercel 하나로 통합**하는 것이 간단합니다.
