# 농식품 주요통계 대시보드

농림축산식품부 『농림축산식품 주요통계 2025』(581p)를 일반 공개용 웹 도구로 만드는 프로젝트.
원문이 방대하고 사전처럼 나열돼 활용성이 낮은 문제를, 대시보드·검색·AI 분석 3개 층으로 해결한다.

## 제품 구성 (3개 층)
1. **대시보드** — 핵심지표를 카드+그래프로 한눈에
2. **검색** — 자연어로 원하는 자료 찾기 (구조화 필터 → 이후 RAG)
3. **AI 분석·조언** — 지표를 엮어 인사이트 제공 (실제 수치 근거만, 숫자 생성 금지)

## 핵심 설계 원칙
대시보드·검색·AI가 **같은 JSON 하나**를 읽는다. 지표 1개 = 레코드 1개.
모든 레코드에 `source`(장·페이지) 필수 → 공공누리 출처표시 + AI 인용 근거.

## 개발 순서
데이터 정리 → ① 대시보드 → ② 검색 → ③ AI 분석·조언

## 폴더 구조
```
docs/    설계 문서 (인수인계, 스키마)
data/    지표 JSON (원자료에서 추출)
web/     대시보드·검색·AI 프론트엔드
```
